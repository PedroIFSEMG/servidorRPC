# Sistema RPC Avançado com Interface Gráfica

Um sistema de calculadora distribuída usando Remote Procedure Call (RPC) com interface gráfica desktop.

## Funcionalidades

### 1. Operações Matemáticas
- **Operações básicas**: Soma, subtração, multiplicação, divisão
- **Operações avançadas**: Fatorial, potência, raiz quadrada
- **Resolução de problemas**: Uso da API Gemini para resolver problemas matemáticos complexos

### 2. Sistema de Cache
- Cache no servidor para operações frequentes
- Cache no cliente para melhor desempenho
- Expiração automática de cache

### 3. Interface Gráfica
- Teclado numérico integrado
- Campos dinâmicos baseados na operação selecionada
- Histórico de resultados
- Busca de notícias em tempo real
